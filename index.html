Tugas Pertama - Pengatahuan umum

1) Apa yang kamu ketahui tentang Version Control?
Jawab : Version Control adalah sebuah system yang mencatat setiap perubahan terhadap berkas sehingga suatu saat dapat kembali kepada salah satu versi dari berkas tersebut.

2) Jelaskan apa itu GIT!
Jawab : GIT adalah Version Control System yang digunakan para developer, toper untuk mengembangkan software secara bersama sama.

3)Apa yang kamu ketahui tentang Framwork khususnya web Framework?
Jawab : Aplikasi yang di desain untuk mendukung pengembangan dalam membangun sebuah aplikasi berbasis web dan otomatis web yang kita buat akan terstandarisasi.
contoh framework :
-)PHP FRAMEWORK (Laravel, Code Igniter (CI)
-)CSS FRAMEWORK (Bootstrap)
-)JS FRAMEWORK (Angular-JS)

4)Jelaskan istilah-istilah berikut: REST API, SQL, HTTP, UI Framework?
Jawab :
-) Rest API : Antarmuka yang berupa kumpulan fungsi yang dapat dipanggil atau dijalankan oleh program lain.
-) SQL :Sekumpulan perintah khusus yang digunakan untuk mengakses data dalam database relasional.
-) HTTP : Sebuah protokol jaringan lapisan aplikasi yang digunakan untuk sistem informasi.
-) UI Framework : Framework open source yang bisa diintegrasikan dengan mudah kedalam project yang sedang kita buat.
